Funhouse mirror-exponential function
📐 desmos ready · replace src

transformations

I took this photo in an amusement park in Suzhou, China when my friends and I were playing with a huge distorting mirror. I found the shape of the mirror was closely like a growth exponential function, so I selected the parent function y = 2xas a starting point.Then I noticed this curve is steeper than the shape of the mirror, so I changed the base number from 2 to 1.4 to make the image grow more slowly.Then I temporiraly adjusted the position of my function and found this shape fit the frame of the mirror nicely.After that, I chose the base of the handle as the original point and found my function is below the mirror.So,I did a vertical translation 1.8 units upward and a horizontal translation of 0.5 unit to the right to let the function fit more perfectly.

Then this is the final function I got:
y = 1.4(x - 0.5) + 1.8

Summary of transformations:
• change the base number from 2 to 1.4→ 2xto 1.4x

•vertical translation of 1.8 units upward→1.4x+1.8 •horizontal translation of 0.5 unit to the right→
1.4(x - 0.5) + 1.8 .

Domain and range

The domain of my function is D: {x ∈ ℝ|0 ≤ x ≤ 3.7}. I limited the domain to the portion of the mirror that was visible in the photo.Because the left part of the mirror was covered by my hand and it was invisible, so I limited the minimum x value to 0.The maximum value is just the number at which the mirror frame disappears on the right side.

The parent function y = 2x has a domain of D：{x ∈ ℝ} and a range of R:{y ∈ ℝ|y > 0}.This means this function is defined for all possible x values and can produce any positive numbers output. It also has a horizontal asymptote at y = 0, where the function is undefined.

After applying transformations,my function also has a horizontal asymptote.However, since I restricted the domain to a small interval that lies away from the asymptote, this undefined behavior is not present from my model.Therefore,I do not need to consider the discontinuity when describing the domain and range of this specific situation.

Since the function is continuous on the restricted interval, the range can be determined by evaluating the function at the endpoints of the domain. By evaluating at and x=0 and x=3.7 , I obtained approximate output values of 2.65 and 4.74.

Therefore,the range of the function is approximately R:{y ∈ ℝ|2.65 ≤ y ≤ 4.74}.

Arch-trigonometric function

transformations

I took this photo in front of the Royal Palace of Malaysia in Kuala Lumpur and I found the shape of the arch is like a secant function. So, I chose the parent function y = sec(x) as a starting point. I set the left edge of the arch as the original point and I found the arch is a downward arc, so I had to choose an image within an interval below the y-axis to fit the shape.I decided to choose the nearest part of the function which is in the lower right part of the picture as the beginning point.

I found the curvature of the arch is greater than that of the function, so I increased the period from 2π to approximately 9 by conducting a horizontal stretch by a factor of 1/0.7.Then, I temporarily adjusted my picture and found the curve fit the arch nicely. After that, I did a horizontal translation of 2.83 units to the left and a vertical translation of 2.5 units upward to let this part of the function go to its right place.

The final function I got: y=sec0.7(x+2.83)+2.5

Summary of transformations:
• horizontal stretch by a factor of 1/0.7→ sec(x)→sec(0.7x)
• horizontal translation of 2.83 units to the left→
sec0.7(x+2.83)
• vertical translation of 2.5 units upward→
sec0.7(x+2.83)+2.5

Domain and range

The domain of my function is D:{x ∈ ℝ|0 ≤ x ≤ 3.313}. I chose the left edge of the arch as the starting point, so the minimum value of my domain is zero.I chose 3.313 as the maximum x value is because at this time the y value is closest to zero, which is the most in line with the real situation.

In my parent function y=sec(x),it has many vertical asmyptotes at x = π/2 + kπ(k ∈ Z)After applying transformations, my function also has many vertical asmyptotes. However, since I restricted the domain into a small interval,the asmyptotes became invisible.So I do not need to consider the discontinuity of the domain in this specific situation.

In my parent function, it has a range of (-∞, -1] ∪ [1, ∞).This means that for the part of the function I chose, it has a turning point at -1.After I made a vertical translation of 2.5 units upward,the turning point became 1.5.Because I chose the origin as my starting point,so my minimum y value is zero.

Therefore, the range of my function is {y ∈ ℝ|0 ≤ y ≤ 1.5}

Roof-sinusoidal function
📐 desmos ready · replace src

transformations

I took this picture in Yu garden,Shanghai and I found the roof behind me in this picture is like a cosine function because it repeats inself.So I chose the parent function y=cos(x) as a starting point. I chose my foot as the original point and I found that the curve of the roof as a whole appears much smaller than the function. So I decreased the amplitude and the period of the function by conducting a vertical stretch by a factor of 0.39 and a horizontal compression by a factor of 1/2.39. Then I drew my picture downward and I found the function fit the shape of the roof nicely. After that, I did a horizontal translation of 0.3 unit to the left and a vertical translation of 3.6 units upward to let the function come to its right place.

The final function I got :y=0.39cos(2.39(x+0.3))+3.6

Summary of transformations:
• vertical stretch by a factor of 0.39→cos(x)→0.39cos(x)
• horizontal stretch by a factor of 1/2.39→0.39cos(2.39x)
• horizontal translation of 0.3 unit to the left→
0.39cos(2.39(x+0.3))
• vertical translation of 3.6 units upward→
0.39cos(2.39(x+0.3))+3.6

Domain and range

Because in my picture, only portion of the roof was visible, so I limited my domain to {x ∈ ℝ|-0.7 ≤ x ≤ 1.5} to fit that part.
In my parent function y=cos(x), it has a range of [-1, 1], after I did vertical translations of 3.6 units upward and changed the amplitude from one to 0.39, the maximum y value can be caculated by 0.39+3.6=3.99 and the miminum point is -0.39+3.6=3.21

Therefore, the range of my function is{y ∈ ℝ|3.21 ≤ y ≤ 3.99}
