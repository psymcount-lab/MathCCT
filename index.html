Funhouse mirror-exponential function
📐Amy

transformations

I took this photo in an amusement park in Suzhou, China when my friends and I were playing with a huge distorting mirror. I found the shape of the mirror was closely like a growth exponential function, so I selected the parent function y = 2xas a starting point.Then I noticed this curve is steeper than the shape of the mirror, so I changed the base number from 2 to 1.4 to make the image grow more slowly.Then I temporiraly adjusted the position of my function and found this shape fit the frame of the mirror nicely.After that, I chose the base of the handle as the original point and found my function is below the mirror.So,I did a vertical translation 1.8 units upward and a horizontal translation of 0.5 unit to the right to let the function fit more perfectly.

Then this is the final function I got:
y = 1.4(x - 0.5) + 1.8

Summary of transformations:
• change the base number from 2 to 1.4→ 2xto 1.4x

•vertical translation of 1.8 units upward→1.4x+1.8 •horizontal translation of 0.5 unit to the right→
1.4(x - 0.5) + 1.8 .

Domain and range

The domain of my function is D: {x ∈ ℝ|0 ≤ x ≤ 3.7}. I limited the domain to the portion of the mirror that was visible in the photo.Because the left part of the mirror was covered by my hand and it was invisible, so I limited the minimum x value to 0.The maximum value is just the number at which the mirror frame disappears on the right side.

The parent function y = 2x has a domain of D：{x ∈ ℝ} and a range of R:{y ∈ ℝ|y > 0}.This means this function is defined for all possible x values and can produce any positive numbers output. It also has a horizontal asymptote at y = 0, where the function is undefined.

After applying transformations,my function also has a horizontal asymptote.However, since I restricted the domain to a small interval that lies away from the asymptote, this undefined behavior is not present from my model.Therefore,I do not need to consider the discontinuity when describing the domain and range of this specific situation.

Since the function is continuous on the restricted interval, the range can be determined by evaluating the function at the endpoints of the domain. By evaluating at and x=0 and x=3.7 , I obtained approximate output values of 2.65 and 4.74.

Therefore,the range of the function is approximately R:{y ∈ ℝ|2.65 ≤ y ≤ 4.74}.

I found the curvature of the arch is greater than that of the function, so I increased the period from 2π to approximately 9 by conducting a horizontal stretch by a factor of 1/0.7.Then, I temporarily adjusted my picture and found the curve fit the arch nicely. After that, I did a horizontal translation of 2.83 units to the left and a vertical translation of 2.5 units upward to let this part of the function go to its right place.

The final function I got: y=sec0.7(x+2.83)+2.5

sec0.7(x+2.83)
• vertical translation of 2.5 units upward→
sec0.7(x+2.83)+2.5

Domain and range

The domain of my function is D:{x ∈ ℝ|0 ≤ x ≤ 3.313}. I chose the left edge of the arch as the starting point, so the minimum value of my domain is zero.I chose 3.313 as the maximum x value is because at this time the y value is closest to zero, which is the most in line with the real situation.

Pillar structure - Quadratic Reciprocal Function
📐 sarah

Transformations

I chose this photo as it is the first major structure seen when I first came to Sunway. I established the edge of the leftmost tile on the ground as the origin. The height of the inner curve of the pillar (m) was my vertical scale and the horizontal distance from the leftmost edge of the tile (m) was my horizontal scale. I scaled the photo to match an estimate of the distances in meters.

To model the function onto the photo, I began by identifying a shape that most closely resembled a quadratic reciprocal shape, which was the inner curve of the pillar structure. I used the parent function \(y=\frac{1}{x^2}\) as a starting point. As the orientation of the inner curve faces downwards, I chose to reflect the parent function by the x-axis, modifying the function to \(y=-\frac{1}{x^2}\).

As the inner curve of the pillar structure is significantly smaller than the initial parent function, I horizontally compressed my modified function by a factor of \(\frac{1}{3.16}\), leaving me now with the function \(y = -\frac{1}{(3.16x)^2}\). Once the shape was fixed, I aligned it to the inner curve using both horizontal and vertical translations. I shifted the graph to the right by 1.95 units and then shifted it upwards by 20 units . This resulted in the function \(y=-\frac{1}{(3.16(x-1.95))^2}+20\), which closely models the inner curve of the pillar structure in my photo.

Final function: \(y=-\frac{1}{(3.16(x-1.95))^2}+20\)

Summary of transformations:
• Reflection on x-axis: \(y = \frac{1}{x^2} \rightarrow y = -\frac{1}{x^2}\)
• Horizontal compression of factor \(\frac{1}{3.16}\): \(-\frac{1}{(3.16x)^2}\)
• Horizontal translation of 1.95 units to the right: \(-\frac{1}{(3.16(x-1.95))^2}\)
• Vertical translation of 20 units upwards: \(-\frac{1}{(3.16(x-1.95))^2}+20\)

Domain and Range

D : { \(x \in \mathbb{R} \mid 1 \le x \le 3, x \ne 1.95\) }
R : { \(y \in \mathbb{R} \mid 16 \le y < 20\) }

The parent function \(y = 1/x^2\) has a domain of D : { \(x \in \mathbb{R} \mid x \ne 0\) } and a range of R : { \(y \in \mathbb{R} \mid y > 0\) }. It has a vertical asymptote at \(x = 0\) and a horizontal asymptote at \(y = 0\), where the functions are undefined.

The domain of my function is D : { \(x \in \mathbb{R} \mid 1 \le x \le 3, x \ne 1.95\) }. I limited the domain to the portion between the middle points of the two inner curves on the pillar structure [1,3], as that portion most closely resembled a quadratic reciprocal function. My function has vertical asymptote at \(x = 1.95\), as a result of my shift of the function 1.95 units to the right, aligning with the pillar in between the two inner curves. While the parent domain of the function only excludes the asymptote, I applied a restricted domain of [1,3] to isolate the specific section of the pillar being modeled, resulting in a final domain of D : { \(x \in \mathbb{R} \mid 1 \le x \le 3, x \ne 1.95\) }.

The range of my function is R : { \(y \in \mathbb{R} \mid 16 \le y < 20\) } I limited my domain to [16,20) as that image portion most closely models a quadratic reciprocal function at that restricted range. The graph opens down, due to a reflection on the x-axis from my parent function, so \(y < 20\) is used. My function has a horizontal asymptote at \(y = 20\), so the symbol < is used.

Painted circle - Logarithmic Function
📐 Sarah

Transformations

I chose this photo as I walk past it occasionally when going to the uni library. I established the outer right door corner as the point of origin. The height of the painted circle (m) was my vertical scale and the horizontal distance from the outer right door corner (m) was my horizontal scale. I scaled the photo to match an estimate of the distances in meters (m).

To model the function onto the photo, I began by identifying the shape that most closely resembled a logarithmic function, which was the top left side of the painted circle. I used the parent function \(y= \log_2(x)\), as a starting point.

As the curve of the painted circle is less steep than the gradient of the initial parent function, I horizontally compressed the function by a factor of \(\frac{1}{4.1}\), leaving me with the function \(y=\log_2(4.1x)\). Next I noticed that this made the function slightly too steep to be modelled onto the painted circle, so I vertically compressed it by a factor of \(\frac{1}{2}\), leaving me now with the function \(y=\frac{1}{2}\log_2(4.1x)\). Once the shape was fixed, I aligned it with the curve of the painted circle using both horizontal and vertical translations. I shifted the function 2.43 units to the left, then shifted it 1.2 units upwards. This resulted in the function \(y=\frac{1}{2}\log_2(4.1(x+2.43))+1.2\), which closely models the curve of the painted circle in my photo.

Final Function: \(y=\frac{1}{2}\log_2(4.1(x+2.43))+1.2\)

Summary of transformations:
• Horizontal compression of factor \(\frac{1}{4.1}\): \(\log_2(4.1x)\)
• Vertical compression of factor \(\frac{1}{2}\): \(\frac{1}{2}\log_2(4.1x)\)
• Horizontal translation of 2.43 units to the left: \(\frac{1}{2}\log_2(4.1(x+2.43))\)
• Vertical translation of 1.2 units upwards: \(\frac{1}{2}\log_2(4.1(x+2.43))+1.2\)

Domain and Range

\(D : \{ x \in \mathbb{R} \mid -2.05 \le x \le -1.8 \}\)
\(R : \{ y \in \mathbb{R} \mid 1.52 \le y < 1.88 \}\)

The parent function \(y= \log_2(x)\) has a domain of \(D : \{ x \in \mathbb{R} \mid x > 0\}\) and a range of \(R : \{ y \in \mathbb{R} \}\). It has a vertical asymptote at \(x = 0\), where the function is undefined.

The domain of my function is \(D : \{ x \in \mathbb{R} \mid -2.05 \le x \le -1.8 \}\). I limited the domain to a section of the upper left curve of the painted circle \([-2.05,-1.8]\) as that is the portion where my logarithmic curve most closely fits. As a result, my model is only valid for this portion of the painted circle in my image.

After applying transformations, my function still has a vertical asymptote but since I restricted the domain to the interval \([-2.05,-1.8]\) which lies away from the asymptote, the asymptote is not present in my model. Thus, I don’t need to consider the asymptote when describing the domain in this situation.

Since the function continually increases within the restricted domain interval \([-2.05,-1.8]\), the range can be determined by evaluating the function at these two specific endpoints. By substituting the minimum domain value (\(x = -2.05\)) into the equation, we can determine the lower bound of the range to be approximately 1.52. Similarly, by substituting the upper domain value (\(x = -1.8\)) into the equation, we can determine the upper bound of the range to be approximately 1.88. The range is expressed as \(R : \{ y \in \mathbb{R} \mid 1.52 \le y < 1.88 \}\).

Transformations

I chose this photo as it’s a signboard that I come across often when arriving on campus. I established the bottom left edge of the signboard as the point of origin. The height of the signboard (cm) was my vertical scale and the horizontal distance from the bottom left edge of the signboard (cm) was my horizontal scale. I scaled the photo to match an estimate of the distances in cm.

To model the function onto the photo, I began by identifying the shape that most closely resembled a degree 3 polynomial function, which were the white rings on the signboard. I used the parent function \(y=x^3\) as a starting point.

As the gradient of the curve of the rings was less steep than the gradient of the initial parent function, I horizontally stretched the function by a factor of 3.8, leaving me with the function \(y=(\frac{1}{3.8}x)^3\). Once the shape was fixed, I aligned it with the curve of one of the rings using both horizontal and vertical translations. I shifted the function 19 units to the right, then shifted it 15.5 units upwards. This resulted in the function \(y=(\frac{1}{3.8}(x-19))^3+15.5\), which closely models the curve of one of the rings on the signboard.

Final Function: \(y=(\frac{1}{3.8}(x-19))^3+15.5\)

Summary of transformations:
• Horizontal stretch of factor 3.8 : \((\frac{1}{3.8}x)^3\)
• Horizontal translation of 19 units to the right : \((\frac{1}{3.8}(x-19))^3\)
• Vertical translation of 15.5 units upwards : \((\frac{1}{3.8}(x-19))^3+15.5\)

Domain and Range

\( D : \{ x \in \mathbb{R} \mid 20 \le x \le 23.5 \} \)
\( R : \{ y \in \mathbb{R} \mid 15.52 \le y \le 17.16 \} \)

The parent function \( y=x^3 \) has a domain of \( D : \{ x \in \mathbb{R} \} \) and a range of \( R : \{ y \in \mathbb{R} \} \).

The domain of my function is \( D : \{ x \in \mathbb{R} \mid 20 \le x \le 23.5 \} \). I limited the domain to the lower quarter of one of one of the white rings on the signboard \( [20, 23.5] \) as that is the portion where my degree 3 polynomial curve most closely fits. As a result, my model is only valid for this portion of the signboard in my image.

Since the function continually increases within the restricted domain interval \( [20, 23.5] \), the range can be determined by evaluating the function at these two specific endpoints. By substituting the minimum domain value (\( x = 20 \)) into the equation, we can determine the lower bound of the range to be approximately 15.52. Similarly, by substituting the upper domain value (\( x = 23.5 \)) into the equation, we can determine the upper bound of the range to be approximately 17.16. The range is expressed as \( R : \{ y \in \mathbb{R} \mid 15.52 \le y \le 17.16 \} \).
